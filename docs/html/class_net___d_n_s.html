Net_DNS::opcodesbyname (   $opcode)

Translates opcode names to integers.

Translates the name of a DNS OPCODE into it's assigned number listed in RFC1035, RFC1996, or RFC2136. Valid OPCODES are:

  • QUERY
  • IQUERY
  • STATUS
  • NS_NOTIFY_OP
  • UPDATE
